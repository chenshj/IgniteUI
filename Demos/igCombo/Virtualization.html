Virtualization

Enabling UI virtualization allows the igCombo control to quickly and efficiently render large amounts of data in the combo. When this feature is enabled, the igCombo control reuses HTML elements and replaces the contents of these elements while scrolling which significantly reduces the overhead in rendering new data. This sample accesses the Netflix OData API
