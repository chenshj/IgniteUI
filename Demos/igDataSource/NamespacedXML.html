Namespaced XML

This sample demonstrates how to bind our jQuery Data Source component to XML with namespaces. The difference between namespaced and non-namespaced XML is that with namespaced XML most modern browsers are not able to execute XPath expressions. Therefore the data Source control implements that and allows you to still bind to such data.

Note: In order to properly load the XML data file, this sample must be hosted from a web server along with the javascript, theming, and XML resources.
