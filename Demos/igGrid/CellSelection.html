Cell Selection

This sample demonstrates how to enable cell selection functionality of igGrid control. The Selection feature allows you to select multiple cells in the grid at a time by enabling the multiple selection behavior on the grid. When running in a desktop browser you select cells by clicking on a cell and dragging the selection area with the mouse (because the property ‘mouseDragSelect’ is set to true by default). In the same manner when the property ‘touchDragSelect’ is set to true, you can select multiple cells for touch platforms. Selecting on a touch device, you tap on the first cell you wish to select and tap and swipe through the rest of the selection area.
