Column Summaries Basic Options

This sample demonstrates the basic options you can set for the column summaries of the igGrid control. When a value is entered into the fields below, it is applied to the corresponding column summaries option immediately. The summary option fields will dynamically change to only reflect options that work together. For instance when the dropdown buttons are disabled, the dropdown height, width, and animation options are hidden from view.
