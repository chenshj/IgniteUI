Continuous Virtualization in OutlookGroupBy Modes

This sample demonstrates the continuous virtualization capabilities of the igGrid control in OutlookGroupBy mode. Continuous virtualization is a feature that controls how rows are displayed in the grid. As the user scrolls up and down the grid, the virtualization feature determines if the current rows are sufficient to display the next/previous portion of rows. If new rows are required, then the current portion of rows is disposed and the new portion of rows is created. Therefore a grid with 1,000 records in its data source may only display 30 at a time which significantly reduces the DOM overhead.
