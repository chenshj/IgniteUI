Multiple Sorting And Hiding Events With Modal Window

This sample demonstrates all the igGrid control’s sorting and hiding specific events, along with their parameter arguments. The advanced modal dialog window, for both Sorting and Hiding, allows you to manipulate the igGrid control’s data and structure. By setting the properties ‘modalDialogSortOnClick’ or ‘modalDialogHideOnClick’ to true you immediately apply the action, either hiding columns or sorting one or more rows. If it’s left with the default ‘false’ value, then you can make a series of changes and to apply or reset them at once.
