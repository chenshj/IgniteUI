Row Template with Grid Features

This sample demonstrates how to create a row template using Infragistics Templating Engine with combination of igGrid Paging, Sorting, Filtering and GroupBy features. In this sample, Filtering and GroupBy are not allowed on the first column and the first and the third columns have templates applied.
