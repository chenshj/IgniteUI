Tooltips Popover

This sample demonstrates the new style and behavior of the tooltip. Popover is a style for the grid tooltip that is made for use under touch and desktop applications. The popover is activated when you tap on the touch device screen, respectively click in a desktop application, because ‘hover’ state is not supported for touch devices. Also the popover itself has a close button that allows you to hide the popover when you interact with the grid, using touch device. The difference between to popover and the normal tooltip style is that the popover has touch optimized layout.
