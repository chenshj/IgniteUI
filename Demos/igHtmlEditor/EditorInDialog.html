Editor in Dialog

This sample demonstrates the integration between the igDialog and the igHtmlEditor controls.Press the “Show igHtmlEditor” button to popup dialog which contains an instance of the igHtmlEditorcontrol. An important aspect of this sample is that when you have width and height set to 100% for the editor then the igDialog control should be instantiated first so that the igHtmlEditor can calculate its dimensions correctly.
