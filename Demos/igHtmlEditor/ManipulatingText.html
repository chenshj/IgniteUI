Manipulating Text

This example shows the igHtmlEditor with no additional user interface. You can test the sample by coping text from another web page or Microsoft® Word™ into the content area. You can programmatically add content to the editor by pressing the “Standard Text” button. Once the content is entered, you can select a portion of the content and press the “Emphasize” button to apply a custom style to the selected text. Press the “Print” button to send the contents of the HTML editor to your printer.
