Create Rating Dynamically

This sample demonstrates how to create an instance of our jQuery Rating control dynamically and then destroy it. Destroying a control may be necessary when you need to change read-only options. Read-only options refer to items that once initially rendered may not be changed at a later time. Should you need to change read-only options during user interaction of the control, you have the option to destroy the control and create it again with new values. To destroy the control you can simply call the destroy method.
