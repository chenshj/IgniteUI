All Features Enabled

This sample demonstrates how to load all feature resources files of the control with the Script Loader. When you initialize the Script Loader in JavaScript, resources can be loaded with control name plus dot astrisk (e.g. igHierarchicalGrid.*).

Note: If you initialize a control through MVC Wrappers, all dependent resources are loaded automatically.
