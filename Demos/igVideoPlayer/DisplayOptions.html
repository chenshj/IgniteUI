Display Options

In the following sample you can change the various display options of the video player and observe the effect they have on the control’s rendering. Note: When running this sample on a mobile device, the default device video player controls are rendered, unless the Modernizr JavaScript library is undefined. If for some reason you don’t have the Modernizr library available on your page, then the igVideoPlayer control is rendered, which in some cases may have fewer touch capabilities than the default device video player. That’s why this sample uses the Modernizr library and in mobile contexts the igVideoPlayer falls back to the default device video player.
